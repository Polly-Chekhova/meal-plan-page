# Weekly meal plan (public page)

This repo holds **only** the generated, self-contained meal-plan page (`index.html`), served via
GitHub Pages. It is regenerated from the private `personal-nutritionist` source repo. The page is
protected by a shared client-side login (soft gate — see the source repo's README for the caveat).

Do not edit `index.html` by hand; it is overwritten on each weekly build.
